# Deve Pres
Developer presentation landing page

## Setup

### Backend
```bash
cd backend
npm install
```

### Run backend
```
npm run dev
```

### Backend URL

```text
http://localhost:3000
```

### Frontend
```bash
cd frontend
npm install
```

### Run frontend
```
npm run dev
```

### Frontend URL

```text
http://localhost:5173
```

## Stack
- Vite + TypeScript + SCSS
- Node.js + Express (REST API with CORS)
- Nodemailer (SMTP email sending)
- Dotenv
- TypeScript
